```markdown {"during":1000}
#### Declaring and Using Variables

Variables are used to store information that can be referenced and manipulated in a program. They are created by assigning a value to a name using the equals sign `=`.


```



```python {"type":"code","action":"run","button":"Run"}
my_variable = 10
print(my_variable)
```



```markdown {"during":1000}
In this example, `my_variable` is assigned the value `10`. The `print` function is then used to output the value of `my_variable` to the console.
```



```markdown {"during":1000}
#### Data Types in Python

Python has a variety of data types including integers, floats, strings, and booleans. Understanding these is crucial as they determine what kind of operations you can perform on the data.


```



```python {"type":"code","action":"run","button":"Run"}
integer_example = 5
float_example = 5.0
string_example = 'Hello'
boolean_example = True
print(type(integer_example), type(float_example), type(string_example), type(boolean_example))
```



```markdown {"during":1000}
Here, we declare four variables each with a different data type. The `type` function is used to verify the data type of each variable.
```



```markdown {"during":1000}
#### Real-world Application: Variables in a Shopping Cart

Consider a simple shopping cart system where variables are used to store item prices and quantities. This example demonstrates how variables can be used in calculations.


```



```python {"type":"code","action":"run","button":"Run"}
item_price = 19.99
item_quantity = 3
total_price = item_price * item_quantity
print('Total:', total_price)
```



```markdown {"during":1000}
In this shopping cart example, `item_price` and `item_quantity` are used to calculate the `total_price`, which is then printed to the console.
```

```markdown {"during":1000}
#### Key Concepts Summary

* Python syntax is the set of rules that defines how a Python program is written and interpreted. Remember the importance of indentation!
* Variables are essential in Python, acting as containers for storing data values. Data types like integers, strings, and floats define the kind of data a variable can hold.
* Conditional statements like if, elif, and else allow your programs to execute different actions based on specific conditions, enabling decision-making.
* Data structures such as lists, tuples, and dictionaries help organize, store, and manipulate data efficiently, each with its unique properties and use cases.
* Functions in Python are blocks of reusable code that perform a specific task. Modules are files containing Python code that can be imported and utilized within other Python scripts.
```
