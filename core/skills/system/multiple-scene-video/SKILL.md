# Multiple Scene Video

Use this skill for either:

- create: generate a new multi-scene video from a requirement
- resume: recover an existing multi-scene video run from `output_path`

Keep execution mode-specific details in references:

- Create flow: [references/create.md](references/create.md)
- Resume flow: [references/resume.md](references/resume.md)

Choose `resume` only when the user is clearly asking to continue, recover, or rebuild an existing output directory. Otherwise use `create`.
