# Category: Vibe Coding — Types

Catch-all category for coding-related showcases that don't fit a more specific coding category (e.g., `websites`, `games`, `ai-agents`, `agent-skills`, `mcp-servers`, `platform-dev`).

Primary audience: AI agent learners and builders exploring general-purpose coding workflows.

## When to use this category

Use `vibe-coding` only when no more specific coding category applies. Before picking this category, confirm the showcase is not a better fit for:

- `websites` — building or shipping a website
- `games` — building or remaking a game
- `ai-agents` — constructing chatbots, RAG, or multi-agent systems
- `agent-skills` — authoring Skill Pilot agent skills
- `mcp-servers` — building MCP servers
- `platform-dev` — improving Skill Pilot itself
- `tutorials` / `slides` / `media-generation` — content-generation tasks

If a new pattern recurs here often enough, promote it into its own first-class category.
