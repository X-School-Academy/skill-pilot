# Category: Websites — Types

Build, screenshot, iterate, and ship small websites end-to-end. Pairs with `vibe-coding` and `frontend-design`.

Primary audiences: business owners, AI agent learners, job seekers.

## Types

### W1. One-page site from a brief
- Single landing page from a paragraph of intent. Deployable to a static host.

### W2. Multi-page site
- Marketing or portfolio site with consistent components, nav, and theme.

### W3. Iterative redesign
- Start with screenshots of an existing page, ask the agent for two redesigns, choose, ship.

### W4. Static blog / docs site
- Convert a folder of markdown into a Next.js / Astro / MkDocs site.

### W5. Reverse engineering (site clone)
- Re-implement a target site from a screenshot. (Style and layout only — do not copy proprietary content.)

### W6. Component library starter
- Generate a small reusable design-system kit (button, card, modal, form).
