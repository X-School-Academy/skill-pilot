# Category: Tutorials — Types

Turn raw material into structured, self-paced learning content. Built around `course-creator`.

Primary audiences: AI agent learners, job seekers, teachers.

## Types

### T1. Course from notes
- Convert a folder of notes (markdown, PDFs) into a multi-module interactive course.

### T2. Course from a recording
- Take a meeting recording or screencast, transcribe it, and produce a structured tutorial.

### T3. Course from a repo
- Walk a learner through an existing repository: architecture, key files, exercises.

### T4. Quiz / flashcard pack
- Generate spaced-repetition cards or interactive web quizzes from a topic.

### T5. Graded coding lab
- Starter repo + spec + hidden test harness. The user submits, the agent grades and explains.

### T6. Replayable agent run ("watch me build")
- Capture an agent run and re-narrate it as a watchable tutorial artifact.
