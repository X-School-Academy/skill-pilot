# Category: Games — Types

Remake compact open-source games with PixiJS or three.js. Highly visual, instantly demo-able.

Primary audiences: AI agent learners, students, hobbyists.

## Types

### G1. 2D arcade remake (PixiJS)
- Re-implement a small classic (Snake, Pong, Breakout, Tetris) from a `requirements.md`.

### G2. 3D scene / sandbox (three.js)
- Spinning model viewer, starfield, infinite runner, first-person sandbox.

### G3. Physics-driven mini-game
- Gravity, collision, ragdoll demo — pair with `maths/M4` (calculus + control).

### G4. Reverse engineering from a git URL
- Provide a project git URL, preserve original assets, draft `requirements.md`, ask user to re-implement.

### G5. AI-driven NPC / pathfinding
- Add A* pathfinding or LLM-driven NPC dialogue to an existing toy game.

### G6. Multiplayer prototype
- Two-browser game with a tiny websocket server.
