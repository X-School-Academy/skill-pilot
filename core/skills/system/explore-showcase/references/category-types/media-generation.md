# Category: Media Generation — Types

Generate videos, podcasts, talking avatars, and captioned media.

Primary audiences: content creators, business owners.

## Types

### MG1. Thumbnail / poster batch
- Generate a batch of image variants (YouTube thumbs, social cards, posters) via `create-image-audio`.

### MG2. Short-form video
- Script → scenes → cut → captioned vertical video for TikTok / Shorts / Reels.

### MG3. Long-form video
- 5–10 minute YouTube video from a draft: storyboard, b-roll, captions, chapters.

### MG4. Audiobook / podcast
- Convert a blog or book into a multi-voice audio production with chapter markers.

### MG5. Talking-avatar / lip-sync video
- Drive a still photo or 3D avatar with a script to produce a presenter-style video.

### MG6. Slide-show video
- Combine generated slides + voiceover + transitions into a deck video.

### MG7. Screen recording walkthrough
- Capture a tutorial run and produce a narrated screen recording.
