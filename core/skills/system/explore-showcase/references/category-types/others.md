# Category: Others — Types

Catch-all category for non-coding showcases that don't fit any other category.

Primary audience: anyone using AI agents for tasks outside software development.

## When to use this category

Use `others` only when the showcase is **non-coding** AND no other category fits. Before picking this category, confirm the showcase is not a better fit for:

- `basics` — beginner-friendly first runs
- `browser-tasks` — browser-driven automations
- `tutorials` — turning raw material into learning content
- `slides` — generating decks
- `media-generation` — videos, podcasts, avatars, captions
- `cloud-gpu` — cloud or GPU infrastructure tasks
- `maths` — maths-driven products

If a new pattern recurs here often enough, promote it into its own first-class category.
