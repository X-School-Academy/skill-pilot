# Category: Slides — Types

Generate PDF and HTML decks from notes, ideas, or repositories.

Primary audiences: business owners, job seekers, teachers.

## Types

### S1. Deck from a one-line brief
- "Make me a 10-slide pitch on X" → ready-to-export deck.

### S2. Deck from notes / doc
- Convert a markdown doc or set of notes into a structured deck with speaker notes.

### S3. Deck from a repository
- Auto-generate an "intro to this codebase" deck for onboarding or open-source talks.

### S4. Deck refresh
- Reskin or restructure an existing deck against a new template or audience.

### S5. Data-driven deck
- Pull numbers from a CSV / spreadsheet and embed charts directly in slides.
