# Category: Cloud & GPU — Types

Cloud and GPU infrastructure fluency: AWS EC2, SSH, RunPod GPUs, port-forwarding, model hosting.

Primary audiences: AI agent builders, job seekers entering infra-adjacent roles, business owners scaling apps.

## Types

### CG1. EC2 instance lifecycle
- Launch, configure security group, SSH in, run a command, stop. Pair with `aws-api` and `cloud-operation`.

### CG2. AWS credentials and IAM
- Configure profiles, switch roles, audit an IAM policy.

### CG3. SSH and port forwarding
- Open and manage tunnels via `terminal` with `lifecycle="tmux"`.

### CG4. RunPod GPU pod lifecycle
- Create a pod, SSH in, run a workload, tear down.

### CG5. RunPod serverless endpoint
- Stand up a serverless endpoint, benchmark cold-start and latency.

### CG6. Local model hosting
- Run Ollama / vLLM / ComfyUI locally and route a Skill Pilot agent through it.

### CG7. Cloud cost audit
- Walk an account, surface idle resources, propose a savings plan.
