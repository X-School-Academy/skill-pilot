# Image Prompt Guide

Use direct, concrete visual language. Include the subject, medium, composition, mood, color direction, level of detail, and constraints.

## General Structure

```text
<asset type> for <use case>, <main subject>, <composition>, <style/medium>, <lighting/color>, <background>, <constraints>
```

## Icons

Use `square`.

Prompt pattern:

```text
App icon for <product or feature>, simple memorable symbol of <concept>, centered composition, crisp vector-like 3D render or flat icon style, strong silhouette, limited palette, no small text, clean background
```

Good constraints:

- Ask for a single dominant symbol.
- Avoid tiny labels and text.
- Request high contrast and a clear silhouette.

## YouTube Covers

Use `landscape`.

Prompt pattern:

```text
YouTube cover for <video topic>, <main visual hook>, cinematic landscape composition, large clean negative space for title text, high contrast lighting, vivid but professional colors, sharp focal point, no embedded text
```

Good constraints:

- Leave intentional space for title overlays.
- Avoid generated words unless the final artwork should contain text.
- Make the subject recognizable at small sizes.

## Product Mockups

Use `landscape` for web/product hero art and `portrait` for mobile-first assets.

Prompt pattern:

```text
Product mockup for <product>, <device or object>, realistic materials, clean studio lighting, subtle environment, clear view of the interface area, premium commercial style, no unreadable text
```

## Educational Illustrations

Choose ratio based on placement.

Prompt pattern:

```text
Educational illustration explaining <concept>, clear main subject, simple supporting elements, friendly modern editorial style, uncluttered background, accurate visual metaphor, no dense text
```

## Social Graphics

Use `square` for feed posts, `portrait` for stories and reels covers, and `landscape` for headers.

Prompt pattern:

```text
Social graphic for <platform and campaign>, <main subject>, bold composition, readable empty space for caption overlay, brand-aligned colors, polished commercial design, no embedded text
```

## Quality Checklist

- The prompt names the real target use case.
- The prompt specifies ratio and composition.
- Important text is requested as empty space unless exact lettering is required.
- The subject can be understood at thumbnail size.
- The style is specific enough to avoid generic stock imagery.
