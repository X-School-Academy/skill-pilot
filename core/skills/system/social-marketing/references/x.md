# Twitter/X Platform Guide (Experimental)

## UI Navigation & Elements
- **Home URL**: `https://x.com/home`
- **Post Box**: Role `textbox`, "What is happening?!".
- **Post Button**: Button with text "Post".

## Posting Strategy
- **Threads**: Best for high engagement. Start with a hook, number the tweets (1/n).
- **Tone**: Fast-paced, punchy, "hot takes".
