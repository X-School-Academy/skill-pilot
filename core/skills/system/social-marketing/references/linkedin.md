# LinkedIn Platform Guide

## UI Navigation & Elements
- **Feed URL**: `https://www.linkedin.com/feed/`
- **Post Trigger**: Look for a button or container with text "Start a post".
- **Post Modal**:
  - **Textarea**: Role `textbox`, usually labeled "What do you want to talk about?".
  - **Post Button**: Button with text "Post" (usually bottom right of the modal).
- **Engagement**:
  - **Comments**: Found under each post.
  - **Reply Box**: Role `textbox`, "Add a comment...".

## Posting Strategy
- **Hook**: 1-2 sentences that stop the scroll.
- **Body**: Use bullet points and whitespace.
- **CTAs**: Ask a question to spark comments (LinkedIn's algorithm prioritizes comments).
- **Hashtags**: 3-5 max.
