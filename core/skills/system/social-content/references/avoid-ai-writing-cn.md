# Anti-AI Writing Reference — Chinese (中文)

Chinese AI-generated text has its own distinct set of patterns. This reference covers the vocabulary, structural, and communication tells specific to Chinese-language social media content. Apply these rules for all Chinese posts on Xiaohongshu, WeChat, Weibo, and any Chinese-language LinkedIn content.

## Two-Pass Audit (Chinese)

Same process as English: Pass 1 — write and scan. Pass 2 — rewrite and verify. The rewrite threshold is the same: 5+ vocabulary flags + 3+ pattern categories + uniform rhythm = full rewrite.

## Tier 1 — Always Replace (永远替换)

These phrases are near-universal Chinese AI tells. Never use them.

| Phrase | Issue | Replacement |
|--------|-------|-------------|
| 总而言之 / 综上所述 / 总的来说 | AI closing formula | Cut — just state the conclusion, or use a short conversational wrap-up |
| 值得注意的是 / 值得一提的是 | AI emphasis marker | Cut — the fact should speak for itself |
| 在当今时代 / 随着...的发展 / 近年来 | AI opening formula | Cut — lead with the actual point, not era-setting |
| 希望对你有所帮助 / 希望对你有帮助 | Chatbot artifact | Cut entirely |
| 如有疑问请随时联系 / 如有问题欢迎交流 | Chatbot artifact | Cut entirely |
| 让我们一起来看看 / 让我们一起 / 我们来聊聊 | "Let's" equivalent | Cut — just start the content |
| 不可或缺 | Inflated | 重要, 必须, or describe why specifically |
| 极大的 / 巨大的 / 重大的 | Empty intensifier | Give a number, comparison, or cut |
| 众所周知 | Filler | Cut — if it's actually well-known, don't say it |
| 不可思议 / 令人惊叹 | Claimed emotion | Describe what's actually striking about it |
| 深入探讨 / 深入分析 | "Deep dive" equivalent | 聊聊, 说说, 看看 |
| 赋能 | "Empower" equivalent | 帮助, 支持, or describe the mechanism |
| 抓手 | Corporate buzzword | Name the specific thing |
| 底层逻辑 | Inflated abstraction | Describe the actual logic |
| 闭环 | Buzzword | Describe the actual loop/cycle |
| 颗粒度 | Jargon | 细节 (or just describe the specifics) |
| 引爆 | Inflated | 带动, 推动, or describe with numbers |
| 破圈 | Overused metaphor | Describe what actually happened (reached new audience, went viral, etc.) |
| 天花板 | Overused metaphor | Name the limit specifically |
| 护城河 | Overused metaphor | Describe the actual competitive advantage |

## Tier 2 — Flag When 2+ Cluster (两个以上同时出现则替换)

These are fine individually but become AI tells when stacked.

| Phrase | Replacement |
|--------|-------------|
| 此外 / 另外 / 同时 / 并且 / 而且 | Restructure sentences to avoid stacking conjunctions |
| 首先...其次...最后... / 第一...第二...第三... | Vary the structure. Use natural progression, not numbered lists |
| 不仅...而且... / 既...又... | Simplify to one clause. Pick the stronger half |
| 通过...实现... / 以...为... | Rewrite as active sentence with a concrete verb |
| 基于... / 根据... | Use "从...来看" or restructure without the frame |
| 对于...来说 / 从...角度 | Cut the frame, state the point directly |
| 一定程度 / 某种意义上 | Cut — hedge words that add nothing |
| 相对来说 / 相比之下 | Cut the hedge, make the comparison directly |
| 越来越 / 日益 | Describe the trend with a number, or use a fresher phrase |
| 充分发挥 / 充分利用 | 用好, 发挥, or describe the specific action |

## Tier 3 — Flag Only at High Density (高频出现才标记)

| Phrase | Replacement |
|--------|-------------|
| 丰富 / 多样化 | Name the variety specifically |
| 提升 / 提高 | Give a number, or use a more concrete verb |
| 优化 / 完善 | Describe the specific change |
| 打造 / 构建 | 做, 建, 写, or a more specific verb |
| 聚焦 / 专注 | Say what you're actually working on |
| 持续 / 不断 | Cut — implied by context in most cases |
| 全新 / 最新 | 新 (just "new"), or describe what changed |

## Tier 3 Multi-Word Phrases — High Density Flag

| Phrase | Replacement |
|--------|-------------|
| 提供了全方位的解决方案 | Describe the actual solution |
| 实现质的飞跃 | State what improved and by how much |
| 为用户创造价值 | Name the specific value |
| 助力企业数字化转型 | Describe the actual change, for whom, and the result |
| 打造沉浸式体验 | Describe what the user actually sees/does |

## Chinese-Specific Structural Patterns

### Uniform Paragraph Length
Chinese AI produces paragraphs that are consistently 3-4 lines. Real Chinese social writers vary wildly — one-line paragraphs next to five-line blocks. Break up uniformity.

### 四字成语 (Four-Character Idioms)
Using one is natural. Using 2+ in the same short post is an AI tell. Replace with plain speech. "她做事很认真" beats "她做事一丝不苟、精益求精."

### 的-Construction Density
Chinese AI overuses 的 as a connector. If a sentence has 3+ 的, break it into shorter sentences. "我昨天买的那个红色的很贵的包" → "昨天买了个红包。挺贵的。但很喜欢。"

### Over-Formal Register (过于书面化)
Social content should sound spoken. Replace:
- 是否 → 是不是 / 吗
- 如何 → 怎么
- 尚未 → 还没
- 进行 → (often cuttable — "进行讨论" → "聊聊")
- 予以 → 给
- 具有 → 有

### Numbered Lists Overuse
"以下三点建议" / "分享5个方法" — fine occasionally, but if every post has this structure, it's AI. Sometimes just tell the story without numbering.

### Emoji Density
Xiaohongshu uses emojis — that's platform-correct. But Chinese AI places one emoji at the start of every single paragraph like a bullet. Humans scatter them irregularly: sometimes mid-sentence, sometimes none for a whole paragraph, sometimes two in a row.

### 话题标签 (Hashtag) Discipline
- Xiaohongshu: 5-10 tags is fine (platform norm). 15+ is AI stuffing.
- Weibo: 2-5 tags.
- WeChat Moments: Usually zero, maybe one branded tag.

### Title Patterns to Avoid
- "关于XX，你需要知道的N件事"
- "XX终极指南"
- "干货分享｜XX"
- The vertical bar ｜ as a title separator is overused by AI — vary with spaces, 【】, or nothing

## Xiaohongshu Chinese Voice Guide (小红书中文语气)

### Voice
Like sharing tips with a close friend over coffee. Colloquial, warm, personal. Use:
- 我/你 naturally (not 笔者/读者)
- Short sentences (8-15 characters is fine, even 3-5 for punch)
- 语气词 sparingly: 呀, 啦, 哦, 嘛 — one or two per post, not every sentence
- 口语: 超 (instead of 非常), 真的, 太...了, 蛮
- Occasional English loanwords are fine if natural: check, tips, vibe

### Structure (Image+Text Format)
Unlike English platforms, Xiaohongshu posts are typically **image-first with supporting caption**, not the other way around.

**Image Slides (1-9 images):**
- Each image = one key point with styled text overlay
- Slide 1: Title card — the hook, big text, eye-catching
- Slides 2-N: One takeaway per slide. Title line + 2-3 lines of detail
- Last slide: Summary or CTA
- Background: gradient, solid aesthetic color, or high-quality photo — never plain white
- Font: large, readable on phone screen without zooming
- Leave breathing room — don't fill the entire image with text

**Caption (笔记文字):**
- Short (50-150 characters). The images do the heavy lifting.
- Opens with context, not a repeat of the title card
- Ends with 5-10 hashtags
- Can include emojis naturally

### Content Types That Work
- 经验分享 (personal experience) > 知识科普 (educational) > 产品推荐 (recommendations)
- "我试过了" beats "你应该" — personal testimony over generic advice
- Specific numbers build trust: "用了3个月" not "用了一段时间"
- Before/after, mistakes made, money spent — concrete details

## Pre-Presentation Checklist (Chinese Content)

Before showing Chinese content to the user, verify:
- [ ] Zero Tier 1 phrases (总而言之, 值得注意的是, 在当今时代, etc.)
- [ ] No Tier 2 clusters (2+ of 此外/另外/首先/其次/etc. in one paragraph)
- [ ] No chatbot artifacts (希望对你有所帮助, 如有疑问请随时联系)
- [ ] No "让我们一起" constructions
- [ ] Sentence/paragraph length varies (not uniformly 3-4 lines)
- [ ] 的-count: no sentence has 3+ 的
- [ ] 四字成语: 2 max per post
- [ ] Emojis scattered irregularly, not one per paragraph-start
- [ ] Hashtag count matches platform (XHS: 5-10, Weibo: 2-5)
- [ ] If XHS image post: content is structured as slides, not a text wall
- [ ] Reads like a person typing on their phone, not a newspaper article
