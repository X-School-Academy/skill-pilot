# Feature Lifecycle: Implement

Use when an approved `plan.md` (or `update-plan.md` / `issues-plan.md`) is ready for coding.

## Steps

1. Read the relevant plan file (`plan.md`, `update-plan.md`, or `issues-plan.md`).
2. Make the code changes.
3. Verify the implementation with checks appropriate to the touched area.
4. Update `implementation.md` to reflect the current state of what was built.
5. Update `README.md`, `CHANGELOG.md`, and `AGENTS.md` as needed to reflect changes in usage, behavior, or agent instructions.
