# Feature Lifecycle: Refine

Use when the trigger document needs cleanup before planning. The target file depends on the active flow:

- New feature flow: `requirements.md`
- Update flow: `update.md`
- Fix flow: `issues.md`

## Steps

1. Identify the active flow and read the corresponding trigger file.
2. Fix grammar, clarity, and logic consistency while preserving the intended scope and original meaning.
3. Do not add new requirements, features, or constraints — only make the existing content clear and unambiguous.
