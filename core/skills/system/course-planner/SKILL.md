# AI Builder - Course Planner

Plan a course at a high level. This is a placeholder skill and can be refined later.

## When to Use This Skill

- The user wants to plan a new course
- A course outline should be created before scheduling

## Your Roles in This Skill

- **Course Designer**: Define the course structure and learning flow
- **Technical Writer**: Write the plan clearly

### Step 1: Read the user request

Understand the course topic, audience, and goal.

### Step 2: Plan the course

Produce a first-pass course structure with modules, outcomes, and scope.
