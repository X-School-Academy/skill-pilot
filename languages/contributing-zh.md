# 贡献指南

感谢您对 Skill Pilot 项目的兴趣！

## 如何贡献

1. **提交 Issue**
   - 报告错误或提出功能请求
   - 清楚地描述问题和期望的解决方案

2. **创建 Pull Request**
   - Fork 仓库
   - 创建功能分支：`git checkout -b feature/your-feature`
   - 提交更改：`git commit -am 'Add new feature'`
   - 推送到分支：`git push origin feature/your-feature`
   - 创建 Pull Request

3. **代码规范**
   - 遵循项目的代码风格
   - 编写清晰的提交信息
   - 添加必要的测试和文档

## 开发工作流

- 创建分支时使用有意义的名称
- 保持提交历史清晰
- 更新文档反映您的更改

## 许可证

通过贡献，您同意您的贡献将在 MIT 许可证下发布。
