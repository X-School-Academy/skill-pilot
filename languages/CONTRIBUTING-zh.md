# 贡献指南

感谢你对 Skill Pilot 的兴趣！我们欢迎各种形式的贡献。

## 开始贡献

### 1. Fork 仓库

在 GitHub 上点击 "Fork" 按钮来创建仓库的个人副本。

### 2. 克隆你的 Fork

```bash
git clone https://github.com/YOUR_USERNAME/skill-pilot.git
cd skill-pilot
```

### 3. 创建功能分支

```bash
git checkout -b feature/my-feature
```

### 4. 进行更改

进行你想要的代码更改。

### 5. 提交更改

```bash
git add .
git commit -m "feat: add my feature"
git push origin feature/my-feature
```

### 6. 创建 Pull Request

在 GitHub 上创建 Pull Request，描述你的更改。

## 代码标准

- 遵循 PEP 8 风格指南
- 添加适当的测试
- 更新相关文档
- 使用有意义的提交消息

## 报告 Bug

如果你发现 Bug，请创建一个 GitHub Issue，并包含：

- Bug 的清晰描述
- 重现步骤
- 预期行为
- 实际行为
- 你的系统信息

## 提出功能建议

我们欢迎新功能的建议！请创建一个 Issue 来讨论你的想法。
