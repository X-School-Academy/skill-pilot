# 医生消息指南

## 概述

医生消息功能帮助诊断和解决常见问题。

## 常见消息

### 消息：缺少依赖
**原因**：项目依赖未完全安装
**解决方案**：运行 `npm install`

### 消息：配置文件未找到
**原因**：`.env` 文件不存在
**解决方案**：复制 `.env.example` 并配置必要参数

### 消息：端口已被占用
**原因**：服务端口已被其他应用占用
**解决方案**：更改 `.env` 中的 PORT 设置或关闭占用端口的应用

## 获取帮助

如果问题未解决，请：
1. 查看完整文档
2. 在 GitHub 创建 Issue
3. 加入社区讨论
