# 设置指南

## 系统要求

- Node.js 14.0 或更高版本
- npm 6.0 或更高版本
- Python 3.7 或更高版本

## 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/X-School-Academy/skill-pilot.git
   cd skill-pilot
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境**
   - 复制 `.env.example` 到 `.env`
   - 填写必要的配置信息

4. **运行项目**
   ```bash
   npm start
   ```

## 故障排除

如遇到问题，请参考[医生消息指南](doctor-message-guide-zh.md)或查看项目文档。
