# Skill Pilot 技能飞行员

Skill Pilot 是一个开源平台，帮助学生探索和掌握各种技能。我们致力于提供高质量的学习资源和实践机会。

## 核心功能

- **技能库**：丰富的技能课程和教程
- **学习追踪**：记录学习进度和成就
- **社区交流**：与其他学习者互动交流
- **资源共享**：共享和发现优质学习资源
